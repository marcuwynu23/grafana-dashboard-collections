# Contributing Guidelines

Thank you for your interest in contributing to this project.

## How to Contribute

- Open an issue to discuss bugs, improvements, or new dashboard ideas.
- Fork the repository and create a focused branch for your work.
- Keep changes clear, minimal, and well documented.
- Open a pull request using the provided template.

## Pull Request Checklist

- [ ] My change has a clear purpose and description.
- [ ] I updated relevant documentation if needed.
- [ ] I tested the dashboard or configuration changes.
- [ ] I followed the project code of conduct.

## Reporting Issues

When reporting a bug, include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment details (Grafana version, data source, browser, etc.)

## Communication

Be respectful and constructive in all interactions.
